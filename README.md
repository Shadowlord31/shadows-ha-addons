# Shadow's Home Assistant Add-ons

Repository of Home Assistant Supervisor add-ons.

## Add-ons in this repository

### [Garden Journal](./garden_journal)
A self-hosted garden journal / garden diary as a Home Assistant add-on. Track beds,
plantings, harvests, perennials, planning, and costs — with matching Lovelace cards
(see [gartentagebuch-cards](https://github.com/Shadowlord31/gartentagebuch-cards)).

## Installation

1. In Home Assistant, go to **Settings > Add-ons > Add-on Store**
2. Click the three-dot menu (top right) > **Repositories**
3. Add: `https://github.com/Shadowlord31/shadows-ha-addons`
4. Find the add-on in the store and install it
